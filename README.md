# sporter
  
 大一时候用vue写的移动端的社交类web app...

## 网页端运行
 npm run dev

  1.注册登录<br>
  2.在线匹配<br>
  3.添加好友<br>
  4.即时通讯(ws<br>
  5.新闻资讯<br>
